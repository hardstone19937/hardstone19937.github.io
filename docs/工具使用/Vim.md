# Vim
> 《太空漫游2001》中，古猿将骨头抛向空中欢呼的瞬间，骨头变成了太空中的空间站。二者本质并没有什么不同不是吗？古猿用骨来制作工具，捕获猎物，人类使用空间站去探索宇宙，在巨大的智慧(假设存在)下，人类和古猿并无二致，沉浸在使用自己的小智慧所创造的工具的喜悦中。  

> 计算机的学习很多时候是在学习新的工具，工具种类繁多，如果要求深入理解所有工具的底层那是需要耗费巨大的功夫的，很多时候都是边学边用，把玩一个黑盒子那样去尝试，在使用的过程中逐渐理解本质。

Vim是Vi的升级，gVim用于图形化界面的情况（有Windows版本，但是用起来感觉差些意思，不如Linux平台）。NeoVim和Vim的设计观念有些不同，可能更像Vscode一些？笔者并没有用过NeoVim不作评价，它使用Lua语言配置而不是Vim原生的Vimscript。
